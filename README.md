# CRM Edge Insights

A professional blog website dedicated to Customer Relationship Management (CRM) insights, trends, and best practices.

## Overview

**CRM Edge Insights** is a modern, minimalist blog platform featuring:

- **Professional Design**: Clean, spacious layouts with a sophisticated color palette (charcoal, cream, and teal accents)
- **Typography-Driven**: Uses Playfair Display for headers and Lato for body text to create visual hierarchy
- **Responsive Layout**: Fully responsive design that works seamlessly on desktop, tablet, and mobile devices
- **Article Management**: Featured articles, recent posts, and categorized content
- **CRM Resources**: Curated tools, guides, and resources for CRM professionals
- **Newsletter Subscription**: Email subscription integration for readers

## Features

- **Hero Section**: Eye-catching introduction with call-to-action buttons
- **Featured Article**: Highlighted article with full details and metadata
- **Article Grid**: Browse recent articles with category tags and reading time estimates
- **Resource Section**: Access to implementation guides, best practices, and ROI calculators
- **Professional Navigation**: Sticky header with easy access to main sections
- **Footer**: Comprehensive footer with links and company information

## Design Philosophy

The site follows a **Modern Minimalist Professional** aesthetic with a charcoal and teal color scheme, sophisticated typography, and subtle micro-interactions.

## GitHub Pages Setup

This repository is configured for GitHub Pages hosting. To enable it:

1. Go to **Settings** → **Pages**
2. Select **Deploy from a branch**
3. Choose **main** branch and **/ (root)** folder
4. Click **Save**

Your site will be published at: `https://robertlalruatpuia138-spec.github.io/CRM-Edge-Insights/`
